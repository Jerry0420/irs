請幫我製作一款線上即時回饋系統，包含兩個網頁：一個是「觀眾投票頁面」，一個是「主持人後台管理頁面」。

主持人後台管理頁面所需功能：
1. 不需要註冊登入即可使用
2. 主持人可以新增修改刪除題目和答案
3. 可以看到目前題目的即時數據與畫面預覽
4. 「切換題目」控制鈕（切換時觀眾端會即時自動同步切換）
5. 「清除數據/重新投票」的按鈕

題目：皆是文字

選項類型:
1. 有 n 個文字選項
2. 只有四張圖片，排成田字

投票結果呈現：
1. 文字雲 -> 當某個選項越多人選擇，該選項的文字尺寸就會即時平滑放大；相反地，較少人選擇的選項，文字尺寸就會比較小。
2. 圖片插針投票 (Clickable Image Map with Pins) -> 觀眾點擊圖片，即可完成投票。該圖片冒出一個「大頭針（Pin）」或小紅點。越多觀眾點擊某個情景主題，該區域的大頭針就會越密集。

觀眾端所需功能：
1. 不需要註冊登入，觀眾輸入網址或掃 QR Code 即可進入
2. 專為手機瀏覽器優化，按鈕與圖片點擊區域要大。每個觀眾在同一題只能投一票（投完後鎖定畫面，顯示「投票成功，請等待下一題」）
3. 視覺風格要求（溫暖人文木質風）：
- 全站色調：背景採用溫暖的柔和米白色（如 #F5F2EB），主要文字與按鈕採用沉穩的深褐木質色（如 #3E2723）。
- 字體：文字與選項請使用優雅且富有藝術感的襯線體（Serif，如明體或宋體），營造的精緻紙質質感。
- 動態效果：文字雲的縮放、大頭針的出現，都必須帶有平滑的 CSS 漸變動畫（Transition）。
